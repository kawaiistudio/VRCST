## 📦 Installation

### Requirements
- Python 3.10+ (3.12 recommended)
- Windows 10/11

### Quick Start
1. Download `VRCST.Ver.2.2.1-Release.zip`
2. Extract to any folder
3. Run `RunMe.bat` (auto-installs dependencies)
4. Login with your VRChat credentials

### Manual Installation
```bash
pip install -r requirements.txt
python VRCST.py
```
